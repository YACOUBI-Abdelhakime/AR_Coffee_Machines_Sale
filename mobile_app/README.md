# KawaApp

### This application integrates augmented reality to show the products of the company PayeTonKawa

## Getting Started

#### 1. Import all packages

```bash
flutter pub get
```

#### 2. Launch project

##### with null safety

```bash
flutter run
```

##### without null safety if you have an error

```bash
flutter run --no-sound-null-safety
```

#### 3. Start tests

```bash
flutter test
```
